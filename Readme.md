# GeoData App

Este proyecto es una aplicación para gestionar datos geográficos.

## Estructura del Proyecto

```
geodata-app/
├── assets/         # Imagenes, fuentes, etc.
├── src/
│   ├── components/ # Componentes reutilizables
│   ├── screens/    # Paginas principales de la app
│   ├── navigation/ # Configuracion de la navegacion de la app
│   └── styles/     # Estilos globales
│   
├── public/
│   ├── index.html
│   └── favicon.ico
├── .gitignore
├── package.json
├── App.js
└── README.md
```

## Instalación

1. Clona el repositorio:
    ```sh
    git clone https://github.com/tu-usuario/geodata-app.git
    ```
2. Navega al directorio del proyecto:
    ```sh
    cd geodata-app
    ```
3. Instala las dependencias:
    ```sh
    npm install
    ```

## Uso

Para iniciar la aplicación en modo de desarrollo:
```sh
npm start
```
En la misma terminal se daran mas opciones para visualizar el proyecto de una manera adecuada, dependiendo de lo que se tenga instalado.
